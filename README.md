2.清除浮动：对页面浮动和清除浮动有了一定的认识。<br />
3.对于之前自己不太熟悉的dl dt dd元素有了一定的运用和熟悉。<br />
4.对于块状元素和内联元素的转行比较熟悉。<br />
5.在页面布局的宏观性上还需要进一步把握和清晰。<br />
6.在保持静态页面练习的基础上，开始学习frameset以及动态页面入门。
